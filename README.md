imgseqRepair
===========

*Repairs image sequences by duplicating frames*

## Instructions

### Check for dropped frames
    ./imgseqRepair <path>
### Repair dropped frames
    ./imgseqRepair -d <path>

## Requirements
* [node.js](http://nodejs.org/)
